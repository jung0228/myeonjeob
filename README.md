# myeonjeob

대학원 면접 준비를 위한 개인 정리용 저장소입니다.

## 구성

- `index.html`: 면접 준비용 메인 페이지
- `styles.css`: 페이지 스타일
- `app.js`: 예상 질문 필터 인터랙션
- `docs/linear-algebra-interview-notes.md`: 선형대수 면접 질문, 취약 개념, 모범 답안
- `docs/self-introduction.md`: 자기소개 정리 템플릿
- `docs/research-plan.md`: 연구계획 정리 템플릿
- `docs/mock-interview-notes.md`: 모의면접 기록 템플릿

## 사용 방법

1. `index.html`을 브라우저에서 열어 전체 질문 구조를 훑습니다.
2. `docs/linear-algebra-interview-notes.md`처럼 과목별 취약 개념 노트를 누적합니다.
3. `docs` 폴더 문서를 채우면서 자기소개와 연구계획을 구체화합니다.
4. 실제 면접 답변은 짧은 버전과 긴 버전 두 가지로 정리합니다.
