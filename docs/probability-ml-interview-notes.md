## 1. 독립과 비상관

### 독립

독립은 한 변수의 값이 다른 변수의 분포에 영향을 주지 않는다는 뜻입니다.

- 사건 기준: `P(A ∩ B) = P(A)P(B)`
- 확률변수 기준: 결합분포가 주변분포의 곱으로 분리됨

직관:

- 서로 아무 정보도 주지 않음
- 하나를 알아도 다른 하나에 대한 불확실성이 줄지 않음

### 비상관(uncorrelated)

비상관은 공분산이 0이라는 뜻입니다.

`Cov(X, Y) = 0`

즉 선형적인 관계가 없다는 뜻이지, 관계 자체가 전혀 없다는 뜻은 아닙니다.

### 둘의 관계

- 독립이면 비상관
- 비상관이라고 독립은 아님

핵심 직관:

- 독립 = 완전 분리
- 비상관 = 선형 관계만 없음

대표 반례:

- `Y = X²`
- 이 경우 `Y`는 `X`에 강하게 의존하므로 독립이 아니지만, 대칭적인 경우 공분산은 0일 수 있습니다.

면접용 한 문장:

독립은 더 강한 개념이고, 비상관은 공분산이 0이라는 약한 조건입니다. 독립이면 비상관이지만, 비상관이라고 해서 일반적으로 독립인 것은 아닙니다.

## 2. 공분산과 공분산행렬

### 공분산

공분산은 두 변수가 평균에서 얼마나 함께 움직이는지를 나타냅니다.

`Cov(X, Y) = E[(X - E[X])(Y - E[Y])]`

직관:

- 같이 커지고 같이 작아지면 양수
- 한쪽이 커질 때 다른 쪽은 작아지면 음수
- 그런 경향이 없으면 0 근처

또는 다음 형태로도 기억해두면 좋습니다.

`Cov(X, Y) = E[XY] - E[X]E[Y]`

### 독립이면 왜 공분산이 0인가

독립이면 결합분포가 분리되므로

`E[XY] = E[X]E[Y]`

따라서

`Cov(X, Y) = E[XY] - E[X]E[Y] = 0`

### 공분산행렬

공분산행렬은 여러 변수의 분산과 공분산을 한 번에 모은 행렬입니다.

- 대각 원소: 각 변수의 분산
- 비대각 원소: 변수들 사이의 공분산

왜 중요한가:

- PCA의 핵심 입력
- 다변량 정규분포의 핵심 파라미터
- 변수 간 선형 관계를 한 번에 요약

### 공분산행렬이 왜 PSD인가

임의의 벡터 `v`에 대해

`v^T Σ v = Var(v^T X)`

입니다.

분산은 항상 0 이상이므로, 공분산행렬은 PSD입니다.  
따라서 공분산행렬의 고유값은 음수가 될 수 없습니다.

## 3. Gaussian distribution

Gaussian distribution, 즉 정규분포는 평균을 중심으로 종 모양을 이루는 분포입니다.

핵심 파라미터:

- 평균 `μ`: 중심
- 분산 `σ²`: 퍼짐 정도

면접용 포인트:

- 많은 자연현상과 노이즈 모델에 자주 등장
- CLT와 연결됨
- ML에서는 Gaussian noise 가정과 MLE에서 자주 사용

## 4. CLT와 대수의 법칙

### 대수의 법칙(LLN)

표본평균이 샘플 수가 커질수록 진짜 평균으로 수렴한다는 정리입니다.

핵심:

- "평균이 맞는 값으로 간다"

### 중심극한정리(CLT)

독립이고 같은 분포를 따르는 확률변수들의 표본평균은, 표본 수가 충분히 크면 원래 분포와 상관없이 정규분포에 가까워진다는 정리입니다.

핵심:

- "평균의 분포가 정규분포처럼 된다"

### 둘의 차이

- LLN: 어디로 가는가를 말함
- CLT: 얼마나 흔들리며 가는지, 그 분포가 어떤 모양인지 말함

면접용 한 문장:

대수의 법칙은 표본평균이 기댓값으로 수렴한다는 정리이고, 중심극한정리는 그 표본평균의 오차를 적절히 정규화하면 정규분포에 가까워진다는 정리입니다.

## 5. Bayes rule

베이즈 룰은 새로운 증거를 관측했을 때 가설의 확률을 업데이트하는 공식입니다.

`P(A|B) = P(B|A) P(A) / P(B)`

용어:

- `P(A)`: prior
- `P(B|A)`: likelihood
- `P(A|B)`: posterior

핵심 직관:

- 원래 얼마나 흔한가(prior)
- 지금 본 증거가 그 가설에서 얼마나 그럴듯한가(likelihood)
- 이 둘을 합쳐서 업데이트된 믿음(posterior)을 만든다

자주 하는 실수:

- `P(A|B)`와 `P(B|A)`를 같은 것으로 착각

면접용 한 문장:

베이즈 룰은 어떤 증거를 본 뒤 가설의 확률을 업데이트하는 공식으로, prior와 likelihood를 결합해 posterior를 계산하는 방법입니다.

## 6. likelihood와 MLE

likelihood는 어떤 파라미터나 가설이 주어졌을 때, 지금 관측한 데이터가 얼마나 그럴듯한지를 나타내는 값입니다.

보통 다음처럼 씁니다.

`P(data | θ)`

같은 식이라도 해석이 다릅니다.

- probability 관점: 파라미터 고정, 데이터가 나올 확률
- likelihood 관점: 데이터 고정, 어떤 파라미터가 더 그럴듯한지 평가

MLE(Maximum Likelihood Estimation)는 likelihood를 최대화하는 파라미터를 찾는 방법입니다.

`argmax_θ P(data | θ)`

## 7. least squares와 Gaussian noise 아래 MLE

선형회귀 모델을

`y = Ax + ε`

로 두고, 오차가

`ε ~ N(0, σ²I)`

라고 가정하면, 관측 데이터의 likelihood는 대략

`P(y|x) ∝ exp(- ||y - Ax||² / (2σ²))`

꼴이 됩니다.

따라서 likelihood를 최대화하는 것은

`||y - Ax||²`

를 최소화하는 것과 같고, 결국 least squares와 MLE가 같은 문제가 됩니다.

면접용 한 문장:

## 체크 문항 상세 해설

### 왜 독립이면 공분산이 0인가

공분산은

`Cov(X,Y) = E[XY] - E[X]E[Y]`

입니다. 독립이면 결합분포가 주변분포의 곱으로 분리되므로

`E[XY] = E[X]E[Y]`

가 됩니다. 따라서 공분산은 0입니다.

직관:

- 독립이면 `X`를 알아도 `Y`가 커질지 작아질지에 대한 정보가 늘지 않음
- 그래서 평균적으로 같이 움직이는 효과가 사라짐

주의:

- 공분산 0이라고 해서 독립은 아님
- `Y = X²`처럼 비선형 관계는 남을 수 있음

### 공분산행렬이 왜 PSD인가

공분산행렬 `Σ`에 대해 임의의 벡터 `v`를 잡으면

`v^T Σ v = Var(v^T X)`

가 됩니다. 여기서 `v^T X`는 여러 변수를 한 방향으로 눌러 만든 1차원 확률변수이고, 분산은 항상 0 이상이므로

`v^T Σ v >= 0`

입니다. 따라서 공분산행렬은 PSD입니다.

여기서 `v^T X`가 뭔가:

- 만약 `X = [X1, X2]^T`, `v = [a, b]^T`라면 `v^T X = aX1 + bX2`
- 즉 여러 변수를 가중합해서 만든 새로운 1차원 확률변수
- 그래서 `v^T Σ v`는 그 방향으로 봤을 때의 분산이라고 읽으면 됨

2차원 식으로 보면:

- `Σ = [[Var(X1), Cov(X1,X2)], [Cov(X1,X2), Var(X2)]]`
- `v = [a,b]^T`일 때
- `v^T Σ v = a²Var(X1) + 2abCov(X1,X2) + b²Var(X2)`
- 이 값이 정확히 `Var(aX1 + bX2)`와 같음

왜 중요한가:

- 공분산행렬의 고유값이 음수가 될 수 없음
- PCA에서 고유값을 각 방향의 분산량으로 해석 가능

### 왜 least squares가 Gaussian noise 아래 MLE인가

모델을

`y = Ax + ε`

로 두고, 오차가

`ε ~ N(0, σ²I)`

라고 가정하면 likelihood가

`P(y|x) ∝ exp(- ||y - Ax||² / (2σ²))`

꼴이 됩니다.

따라서 likelihood를 최대화하는 것은 결국 제곱오차

`||y - Ax||²`

를 최소화하는 것과 같습니다.

직관:

- Gaussian noise는 오차가 작을수록 더 그럴듯하다고 보는 모델
- 그래서 MLE와 least squares가 같은 문제가 됨

### L1과 L2 regularization 차이

- `L1`: `Σ |w_i|`
- `L2`: `Σ w_i²`

직관:

- `L1`은 일부 가중치를 0으로 보내 sparse한 해를 유도
- `L2`는 가중치를 전체적으로 작게 줄이는 shrinkage

예시:

- feature가 많고 중요한 것만 골라야 할 때는 L1이 더 직관적
- 전체 feature를 남기되 과한 가중치만 줄이고 싶으면 L2가 더 자연스러움

### PCA 차원축소 과정

설명 순서는 아래처럼 잡으면 안정적입니다.

1. 데이터를 중심화한다
2. 공분산행렬을 만든다
3. 고유값과 고유벡터를 구한다
4. 고유값이 큰 방향만 남긴다
5. 그 축으로 데이터를 투영한다

핵심:

- 고유벡터 = 주성분 방향
- 고유값 = 그 방향의 분산량

같은 내용은 중심화된 데이터행렬의 SVD로도 설명할 수 있습니다.

왜 중심화하나:

- PCA는 평균 위치가 아니라 순수한 퍼짐 방향을 보고 싶기 때문
- 평균을 빼지 않으면 데이터가 어디에 놓여 있는지와 어떻게 퍼져 있는지가 섞여버림

안 하면 어떻게 되나:

- 데이터가 `(10,1), (10,2), (10,3)`처럼 세로 방향으로 퍼져 있어도
- 중심화를 하지 않으면 `x=10`이라는 큰 평균값 때문에 PCA가 실제 퍼짐 방향인 `y`축 대신 `x`축 쪽을 더 중요하게 볼 수 있음

숫자 예시:

- 데이터가 `(1,1), (2,2), (3,3)`이면 평균은 `(2,2)`
- 중심화하면 `(-1,-1), (0,0), (1,1)`
- 공분산행렬은 대략 `[[1,1],[1,1]]`
- 가장 큰 고유벡터는 `[1,1]` 방향
- 즉 데이터가 거의 대각선 방향으로만 퍼져 있으므로 그 축 하나만 남겨 2차원을 1차원으로 줄일 수 있음

### 로지스틱 회귀란

로지스틱 회귀는 이진분류 확률을 모델링하는 방법입니다.

`P(y=1|x) = σ(w^T x + b)`

여기서

`σ(z) = 1 / (1 + e^{-z})`

는 시그모이드 함수입니다.

핵심 포인트:

- 선형결합을 확률값으로 바꿔줌
- 출력이 0과 1 사이이므로 분류 확률로 해석 가능
- 보통 cross-entropy loss를 최소화하도록 학습함

숫자 예시:

- feature가 하나이고 `w=2`, `b=-3`, 입력이 `x=2`라고 하면
- 선형결합은 `z = wx+b = 2×2-3 = 1`
- 시그모이드를 넣으면 `σ(1) ≈ 0.73`
- 따라서 클래스 1일 확률을 약 73%로 예측
- 반대로 `x=0`이면 `z=-3`, `σ(-3) ≈ 0.047`이라 클래스 1 확률은 약 4.7%

Gaussian noise를 가정하면 likelihood가 squared error의 지수 형태로 나타나므로, likelihood 최대화는 squared error 최소화와 같아지고 least squares 해가 MLE와 일치합니다.

## 8. overfitting

오버피팅은 모델이 학습 데이터의 패턴뿐 아니라 노이즈까지 과도하게 학습해서, 학습 데이터에서는 잘 맞지만 새로운 데이터에서는 일반화 성능이 떨어지는 현상입니다.

### 판단

- training 성능은 계속 좋아짐
- validation / test 성능은 나빠지거나 정체

### 원인

- 모델이 너무 복잡함
- 데이터가 부족함
- 노이즈까지 학습함

### 완화 방법

- validation set 활용
- early stopping
- regularization
- dropout
- data augmentation
- 모델 단순화

## 9. validation data가 필요한 이유

validation data는 모델이 학습 데이터에만 과도하게 맞춰지는지를 확인하기 위해 필요합니다.

즉:

- training set: 학습
- validation set: 일반화 성능 확인 및 하이퍼파라미터 조정
- test set: 최종 평가

면접용 한 문장:

validation data는 학습에 직접 쓰지 않은 데이터에서 성능을 확인해 오버피팅 여부를 판단하고, 하이퍼파라미터와 early stopping을 결정하는 데 사용합니다.

## 10. L1 vs L2 regularization

둘 다 loss에 패널티를 추가해서 모델 복잡도를 줄이고 오버피팅을 완화합니다.

### L1 regularization

`λ Σ |w_i|`

특징:

- 일부 가중치를 0으로 보내는 경향
- sparse한 해를 유도

### L2 regularization

`λ Σ w_i²`

특징:

- 가중치를 전반적으로 작게 줄임
- shrinkage 성향

면접용 한 문장:

L1은 절댓값 패널티라 sparse한 해를 만들고, L2는 제곱 패널티라 가중치를 전반적으로 작게 줄이는 경향이 있습니다.

## 11. 지금 기준으로 바로 외울 문장

### 독립 vs 비상관

독립은 더 강한 개념이고, 비상관은 공분산이 0이라는 약한 조건입니다.

### 공분산행렬

공분산행렬은 각 변수의 분산과 변수 간 공분산을 담은 행렬이며, 항상 PSD입니다.

### CLT

중심극한정리는 표본평균의 분포가 표본 수가 커질수록 정규분포에 가까워진다는 정리입니다.

### Bayes rule

베이즈 룰은 새로운 증거를 보고 가설의 확률을 갱신하는 공식입니다.

### least squares = MLE

Gaussian noise 가정 아래 least squares는 최대우도추정과 같습니다.
