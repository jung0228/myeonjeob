# 선형대수 면접 노트

선형대수 면접에서 자주 흔들리는 개념을 한 번에 정리하는 문서입니다.  
목표는 "알고는 있는데 말할 때 꼬이는 상태"를 줄이고, 구술형 답변으로 바로 꺼낼 수 있게 만드는 것입니다.

## 현재 취약 개념

| 번호 | 취약 개념 | 핵심 교정 |
| --- | --- | --- |
| 1 | 고유값/고유벡터 설명 | 고유벡터는 변환 후에도 같은 직선 위에 남는 벡터, 고유값은 그 배율 |
| 2 | 음의 고유값 해석 | 방향이 완전히 보존되는 것이 아니라 같은 직선 위에서 반전될 수 있음 |
| 3 | 대각화 조건 | rank가 아니라 선형독립인 고유벡터가 `n`개 존재해야 함 |
| 4 | 대수적 중복도 vs 기하적 중복도 | 중근 개수와 실제 독립 고유벡터 수가 맞아야 대각화 가능 |
| 5 | SVD 적용 범위 | 직사각 행렬 포함 모든 행렬에 가능, 대각화 불가능한 경우에도 사용 |
| 6 | 특이값 정의 | 특이값은 `A^T A` 고유값의 제곱근 |
| 7 | PCA 핵심 | 데이터가 가장 많이 퍼진 방향을 찾고, 그 축으로 차원을 줄이는 방법 |

## 질문 1. 고유값과 고유벡터

### 면접용 핵심 답변

행렬이 어떤 벡터에 작용하면 보통 방향과 크기가 함께 바뀝니다.  
그런데 어떤 특별한 벡터는 변환을 거친 뒤에도 원래와 같은 직선 위에 남습니다.  
그 벡터가 고유벡터이고, 그때 얼마나 늘어나거나 줄어드는지를 나타내는 스칼라가 고유값입니다.

수식으로는 `Ax = λx`입니다.

### 직관 포인트

- 행렬은 공간을 변형시키는 기계라고 볼 수 있습니다.
- 고유벡터는 그 기계에 넣어도 축이 유지되는 방향입니다.
- 고유값은 그 축 방향으로 얼마나 확대 또는 축소되는지 나타냅니다.
- 고유값이 음수이면 방향이 뒤집히지만, 여전히 같은 직선 위에 있습니다.

### 자주 하는 실수

- "방향이 바뀌지 않는 게 고유값"처럼 고유값과 고유벡터를 섞어 말함
- "방향이 바뀌지 않는다"라고만 해서 음수 고유값의 반전을 놓침

## 질문 2. 행렬의 대각화

### 면접용 핵심 답변

대각화는 행렬을 `A = PΛP⁻¹` 꼴로 표현하는 것입니다.  
여기서 `P`는 고유벡터들로 이루어진 행렬이고, `Λ`는 대각에 고유값이 놓인 행렬입니다.

행렬이 대각화 가능하려면 선형독립인 고유벡터가 `n`개 존재해야 합니다.  
특히 고유값이 모두 서로 다르면 항상 대각화 가능합니다.  
고유값에 중복이 있으면, 각 고유값에 대해 대수적 중복도와 기하적 중복도가 같아야 합니다.

### 왜 유용한가

- 행렬의 거듭제곱 계산이 쉬워집니다.
- 시스템의 안정성을 고유값으로 빠르게 판단할 수 있습니다.
- 복잡한 변환을 독립적인 축 방향의 스케일 변화로 해석할 수 있습니다.

### 꼭 교정할 표현

- `rank` 수만큼 고유벡터가 있으면 된다는 말은 틀림
- 정확한 표현은 "선형독립인 고유벡터가 `n`개 존재"임

## 대수적 중복도와 기하적 중복도

### 한 줄 정의

- 대수적 중복도: 특성방정식에서 어떤 고유값이 몇 번 반복되는가
- 기하적 중복도: 그 고유값에 대해 실제로 존재하는 선형독립 고유벡터의 개수

### 면접용 설명

특성방정식에서 어떤 고유값이 중근으로 나타나더라도, 실제로 그 수만큼 독립적인 고유벡터가 나와야 대각화가 가능합니다.  
즉 중복된 고유값의 개수와, 그에 대응하는 고유공간의 차원이 맞아야 합니다.

## 질문 3. SVD와 PCA

### SVD의 의미

모든 행렬은 `A = UΣV^T` 형태로 분해할 수 있습니다.

- `V^T`: 입력 공간의 축을 다시 맞추는 회전
- `Σ`: 각 축 방향으로 얼마나 늘이거나 줄이는지 나타내는 스케일
- `U`: 출력 공간에서 최종 방향을 맞추는 회전

즉 SVD는 어떤 행렬 변환을 "회전 -> 스케일 -> 회전"의 세 단계로 분해한 것입니다.

### 각 행렬의 정체

- `U`: `AA^T`의 고유벡터들
- `V`: `A^T A`의 고유벡터들
- `Σ`의 대각 원소인 특이값: `A^T A` 고유값의 제곱근

### 자주 하는 실수

- 특이값이 고유값의 제곱이라고 말함
- 정확히는 고유값에 루트를 씌운 값

### PCA와 연결

PCA는 데이터가 가장 많이 퍼진 방향을 찾는 차원 축소 기법입니다.

- 공분산 행렬의 고유벡터는 주성분 방향입니다.
- 고유값은 그 방향의 분산량, 즉 정보량의 크기입니다.
- 큰 고유값에 대응하는 방향만 남기면 중요한 정보를 최대한 유지하며 차원을 줄일 수 있습니다.

데이터 행렬에 직접 SVD를 적용해도 PCA와 연결됩니다.  
이때 주성분 방향은 `V`의 열벡터로 이해할 수 있습니다.

### 면접용 핵심 답변

SVD는 모든 행렬을 회전, 스케일, 회전의 세 단계로 분해하는 방법입니다.  
`U`와 `V`는 각각 출력과 입력 공간의 정규직교기저이고, `Σ`는 각 방향의 중요도를 나타냅니다.  
PCA는 데이터가 가장 많이 퍼진 방향을 찾는 방법이며, 데이터 행렬에 SVD를 적용하면 그 주성분 방향을 얻을 수 있습니다.

## 질문 4. 고유값/고유벡터가 왜 중요한가

### 한 문장 먼저

행렬이라는 복잡한 변환을 가장 단순한 형태로 이해하게 해주는 열쇠이기 때문입니다.

### 세 갈래 전개

1. 계산 효율  
행렬의 거듭제곱이나 반복 변환 계산이 단순해집니다.

2. 시스템의 장기 거동 파악  
고유값의 크기로 수렴, 발산, 안정성을 판단할 수 있습니다.

3. AI와 데이터 분석 응용  
PCA, 추천 시스템, 그래프 분석, 신호 처리 등 다양한 분야에서 기본 도구로 쓰입니다.

## PCA 빠른 암기 버전

PCA는 데이터가 가장 많이 퍼진 방향을 찾고, 그 방향들만 남겨서 차원을 줄이는 방법입니다.  
공분산 행렬의 고유벡터가 축이 되고, 고유값이 그 축의 중요도를 나타냅니다.

## 지금 외워둘 문장

### 고유값/고유벡터

고유벡터는 행렬 변환 후에도 같은 직선 위에 남는 벡터이고, 고유값은 그때의 배율입니다.

### 대각화

대각화 가능 조건은 선형독립인 고유벡터가 `n`개 존재하는 것입니다.

### SVD

SVD는 모든 행렬을 회전, 스케일, 회전으로 분해하는 방법입니다.

### PCA

PCA는 데이터 분산을 최대한 보존하면서 차원을 줄이는 기법입니다.

## 추가 보강 개념

아래는 모의면접과 꼬리질문에서 계속 막혔던 개념들을 따로 보강한 메모입니다.  
정의 하나만 아는 상태가 아니라, 질문이 섞여 들어왔을 때 이어 말할 수 있게 만드는 것이 목표입니다.

### 1. EVD, 대각화, 직교대각화 차이

- 대각화는 `A = PΛP⁻¹`
- 직교대각화는 `A = QΛQ^T`
- 여기서 `Q`는 직교행렬이므로 `Q⁻¹ = Q^T`

핵심은 `EVD`가 곧 직교대각화는 아니라는 점입니다.  
일반 대각화는 고유벡터 행렬 `P`가 가역이면 되지만, 직교대각화는 고유벡터들을 정규직교기저로 잡을 수 있어야 합니다.  
실수 대칭행렬에서는 직교대각화가 가능합니다.

### 2. 스펙트럴 정리

스펙트럴 정리는 실수 대칭행렬 `A`에 대해, 어떤 직교행렬 `Q`와 대각행렬 `Λ`가 존재해서

`A = QΛQ^T`

로 쓸 수 있다는 정리입니다.

여기서 바로 따라오는 결론:

- 실수 대칭행렬의 고유값은 모두 실수
- 서로 다른 고유값에 대응하는 고유벡터는 서로 직교
- 실수 대칭행렬은 항상 직교대각화 가능

### 3. Rank

`rank(A)`는 행렬이 만들어내는 독립적인 방향의 개수입니다.  
정확히는 열공간의 차원, 또는 행공간의 차원입니다.

예를 들어 `2 x 3` 행렬은 열이 3개 있어도 모두 `R²` 안에 있으므로 독립적인 방향은 최대 2개뿐입니다.  
그래서 `2 x 3` 행렬의 rank는 최대 2입니다.

짧게 말하면:

- `rank` = 살아 있는 독립 방향의 수
- `full column rank` = 모든 열이 선형독립
- `full row rank` = 모든 행이 선형독립

### 4. Null space

`null space`는

`Ax = 0`

을 만족하는 모든 벡터 `x`의 집합입니다.

즉 행렬 변환을 거친 뒤 완전히 0으로 사라지는 입력 방향들입니다.

`null(A) = {0}`라는 것은 `Ax = 0`의 해가 영벡터밖에 없다는 뜻이고, 이는 곧 열벡터들이 선형독립이라는 뜻입니다.  
따라서 `null(A) = {0}`이면 `A`는 full column rank입니다.

### 5. Positive Definite / Positive Semi-Definite

- PSD (`Positive Semi-Definite`, 양의 준정부호):
  - 모든 `x`에 대해 `x^T A x >= 0`
- PD (`Positive Definite`, 양의 정부호):
  - 모든 `x != 0`에 대해 `x^T A x > 0`

대칭행렬에서는 아주 깔끔하게:

- PSD ↔ 고유값이 모두 `>= 0`
- PD ↔ 고유값이 모두 `> 0`

중요한 연결:

- `A^T A`는 항상 PSD
- `A`가 full column rank이면 `A^T A`는 PD

왜냐하면

`x^T A^T A x = ||Ax||² >= 0`

이기 때문입니다.

### 6. Least squares

least squares는 `Ax = b`를 정확히 못 맞출 때

`||Ax - b||²`

를 최소화하는 `x`를 구하는 문제입니다.

기하적으로는 `b`를 `Col(A)` 위로 직교 정사영한 점이 `A x_hat`입니다.

핵심 문장:

- 정사영점 `A x_hat`는 하나
- 하지만 그 점을 만드는 계수 `x_hat`는 여러 개일 수 있음
- 계수까지 유일하려면 열들이 선형독립이어야 함

### 7. 잔차와 정규방정식

잔차(residual)는

`r = b - A x_hat`

입니다.

즉 실제 목표 벡터 `b`와 least squares로 얻은 근사값 `A x_hat`의 차이입니다.

최소화 함수는

`f(x) = ||Ax - b||²`

이고, 이를 전개하면

`f(x) = x^T A^T A x - 2 b^T A x + b^T b`

가 됩니다.

미분해서 0으로 두면

`A^T A x = A^T b`

가 나오고, 이것이 정규방정식(normal equation)입니다.

또한 기하적으로는

`A^T r = 0`

즉 잔차가 열공간에 직교한다는 뜻입니다.

### 8. 의사역행렬(pseudoinverse)

의사역행렬은 역행렬이 없을 때 역행렬처럼 가장 비슷한 역할을 하도록 만든 일반화된 개념입니다. 기호는 `A⁺`입니다.

중요한 역할:

- exact solution이 없을 때 least squares 해를 줌
- 해가 여러 개일 때 minimum-norm solution을 줌
- 정사각 가역행렬이면 `A⁺ = A⁻¹`

full column rank인 경우에는

`A⁺ = (A^T A)⁻¹ A^T`

이고, 따라서 least squares 해는

`x_hat = A⁺ b`

로 쓸 수 있습니다.

SVD 관점에서는

`A = UΣV^T  =>  A⁺ = VΣ⁺U^T`

입니다.

### 9. 특이값이 `A^T A` 고유값의 제곱근인 이유

SVD에서

`A v_i = σ_i u_i`

입니다.

양변에 `A^T`를 곱하면

`A^T A v_i = σ_i² v_i`

가 되므로, `v_i`는 `A^T A`의 고유벡터이고 대응하는 고유값은 `σ_i²`입니다.  
따라서 특이값 `σ_i`는 `A^T A` 고유값의 제곱근입니다.

### 10. 실수 대칭행렬이 왜 특별한가

면접에서 이 질문이 나오면 다음 네 개를 같이 말하는 것이 좋습니다.

- 고유값이 모두 실수
- 서로 다른 고유값에 대응하는 고유벡터가 직교
- 스펙트럴 정리에 의해 직교대각화 가능
- 그래서 계산과 해석이 모두 깔끔함

면접용 한 문장:

실수 대칭행렬은 고유값이 모두 실수이고, 고유벡터를 직교정규기저로 잡을 수 있어서 `A = QΛQ^T` 형태의 직교대각화가 가능합니다.
